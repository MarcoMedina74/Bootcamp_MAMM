# Proyecto: Simulación de la Máquina de Galton

## Descripción del programa
Para este programa utilicé dos funciones principales. La primera, simular_canica(), obtiene el destino final de cada una de las 3000 canicas tras cruzar 12 niveles de obstáculos usando random.randint(0,1) para decidir cada paso. La segunda, graficar_resultados(), recibe los datos acumulados y genera un Histograma con Matplotlib, incluyendo etiquetas de ejes y un título para visualizar la distribución normal.
